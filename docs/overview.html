Project Overview

This project implements a simple speech recognition pipeline using MFCC features, vector quantization (VQ), K-means clustering, and discrete Hidden Markov Models (HMMs). The code is purposefully compact to illustrate the core algorithms.

High-level flow

  • Feature extraction (MFCC): audio is converted to MFCC vectors (not included in code). The repository contains `combined.mfcc` which lists MFCC frames.
  • K-means / codebook: cluster MFCC frames into a codebook of vectors (K-means). Code in `kmeans.c` / `main.c` demonstrates this step.
  • Vector Quantization (VQ): map each MFCC frame to nearest codebook index to produce discrete symbol sequences. `vq_batch.c` does this using `codebook.txt`.
  • HMM training & testing: train discrete HMMs per class using Baum–Welch (`hmm.c`) and evaluate with the forward algorithm.

Files of interest

  • `kmeans.c`, `kmeans.h` — K-means implementation.
  • `main.c` — small driver that reads `combined.mfcc` and runs K-means.
  • `vq_batch.c` — maps MFCC files to symbol sequences using `codebook.txt` and writes `hmm//(train|dev).seq` files.
  • `hmm.c`, `hmm.h` — discrete HMM implementation: init, Baum–Welch, forward, training & test runner.
  • `codebook.txt`, `combined.mfcc` — data files (codebook may be precomputed).
